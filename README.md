# zh-world-guide

一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为通用 HTML 资源存放与展示的容器。

## 项目简介

本仓库主要收集、整理并托管各类独立 HTML 页面。这些页面可能包含信息展示、工具界面、学习笔记、实验性页面等多种用途。所有页面均为静态 HTML，可直接通过浏览器访问。

仓库本身不提供搜索、导航或动态内容服务，仅作为文件存储与版本管理的载体。

## 目录说明

仓库根目录下按页面用途或主题组织子目录，每个子目录内包含对应的 HTML 文件及相关资源（如 CSS、JavaScript、图片等）。

- `docs/`：存放文档说明类页面
- `tools/`：存放实用工具类页面
- `experiments/`：存放实验性或测试性页面
- `archive/`：存放已不再更新但保留的旧版本页面

目录结构会随着页面增加而调整，具体内容请参考仓库文件列表。

## 页面归档说明

- 每个 HTML 页面均为独立文件，不依赖外部服务或数据库。
- 页面中可能引用外部资源（如字体库、图标库），请在查看时确保网络连接正常。
- 部分页面可能包含交互功能，但不会收集用户数据或发送网络请求。
- 页面内容可能随时间更新，历史版本可通过 Git 提交记录查看。

## 维护说明

本仓库由维护者不定期更新。维护内容包括：

- 添加新的 HTML 页面
- 修正已有页面中的错误或失效链接
- 清理不再需要的文件

欢迎通过 Issue 提交页面建议或报告问题。Pull Request 请先与维护者沟通，避免重复工作。
